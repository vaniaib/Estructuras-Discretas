1. ¿Cuáles son las principales diferencias entre Haskell y Java?
Que Haskell es un lenguaje funcional, un paradigma de programación que enfatiza el uso de funciones para resolver problemas, mientras que Java es orientado a objetos, otra diferencia es que Haskell tiene una evaluación perezosa y en Java esa evaluación es estricta, otra diferencia es que Haskell tiene un tipado muy fuerte e infiere y Java no.

2. ¿Por qué Haskell no ha alcanzado una adopción significativa en la industria del software?
Porque puede que las personas tiendan a considerarlo como diferente a otros e incluso dificil.



3. Si pudieras realizar un simple analogía entre Git y GitHub, ¿Cuál se te vendría a la mente?
